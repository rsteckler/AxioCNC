# AxioCNC Website

Modern static website for AxioCNC, built with Tailwind CSS.

## Features

- **Responsive Design** - Works beautifully on desktop, tablet, and mobile
- **Dark Mode Support** - Automatically adapts to system preferences
- **Modern UI** - Clean, professional design matching the NextCNC aesthetic
- **Story-Driven** - Tells the journey from cncjs to NextCNC
- **Feature Showcase** - Highlights all the improvements and new features

## Structure

- `index.html` - Main landing page with hero, story, features, and download sections

## Styling

Uses Tailwind CSS via CDN for easy deployment. The design uses:
- Primary accent color: `hsl(24 95% 53%)` (orange)
- Dark mode support with automatic theme detection
- Responsive grid layouts
- Modern typography and spacing

## Deployment

This is a static site that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply upload the `index.html` file and it will work immediately.

## Customization

To customize:
1. Edit `index.html` directly
2. Replace placeholder download links with actual GitHub release URLs
3. Update GitHub repository links
4. Add screenshots or demo images as needed
